## [0.1.0] - TBD

### Added
- Initial release of swipeable navigation host for Compose Multiplatform
- NavCoordinator interface for type-safe navigation
- SwipeableNavHost composable with gesture support
- Navigation stack management (present, navigateTo, onBackPressed, popToRoot, setRoot)
- Material 3 adaptive layout support
- MVI integration via tv.dpal:flowvi
- Predictive back gesture support
- StateFlow-based navigation state observation
